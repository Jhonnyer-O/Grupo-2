## 📝 Descripción
TechStore es una plataforma web estática diseñada para la visualización de periféricos y equipos de computación de alto rendimiento. El objetivo del proyecto es practicar el flujo de trabajo colaborativo mediante Git y el diseño web responsivo con CSS moderno.


## 🚀 Tecnologías
* **HTML5:** Estructura semántica del contenido.
* **CSS3:** Diseño personalizado con Flexbox, CSS Grid y Variables.
* **Git:** Control de versiones para desarrollo grupal.

## 🛠️ Instalación y Uso
1. Clonar el repositorio: `git clone [url]`
2. Abrir el archivo `index.html` en cualquier navegador moderno.
